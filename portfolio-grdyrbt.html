The Greedy Robot

How I learned the Greedy algorithm

For a Data Structures & Algorithms course I took in Fall 2020, I programmed a virtual robot that can find the shortest distance to a treasure located on a 2-dimensional grid. A caveat of this project was that the robot could not move diagonally, only up, down, left, and right.

To build the robot, I used C++ along with object-oriented design knowledge to create efficient code.

Throughout this project, I was able to fully understand and apply the concept of recursion to build an algorithm that calculates the robot's shortest possible moves until it locates the treasure.
