The Merkle Tree

How I learned about cryptographic hashes

In Winter 2020, I developed a merkle tree that contains cryptographic hashes. The goal of the merkle tree is to take input that you cannot compute simply from knowing its output, meaning it is a one-way function.

I built the merkle tree using C++ and background knowledge on the properties of the tree.

Throughout this project, I was able to gain valuable insight into cryptocurrency standards and the importance of making sure that inputs cannot be computed from the output.
