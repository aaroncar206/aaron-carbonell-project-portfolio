My Spotify Clone

A Spotify interface replica

In 2022, I spent my Summer building a Spotify clone that mimics the functionality of the real-life globally used music streaming service.

To build the clone, I utilized React as my framework and popular front-end languages such as Javascript, HTML and CSS.

Throughout this project, I was able to gain valuable insight into the development techniques industries use to build the beautiful UI that Spotify is known for. I also learned the importance of keeping code modular so that you know how each component of your site functions.
